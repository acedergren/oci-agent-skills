## Curation Philosophy

We manually curate A-Team articles because:

1. **Quality Control**: Only include OCI-relevant content
2. **Lightweight**: Link to articles, don't embed full content
3. **Organized**: Group by OCI service for easy discovery
4. **Current**: Links always point to latest version
